INsPeCT - INtegrative Platform for Cancer Transcriptomics

What is INsPeCT?

INsPeCT is an innovative data science tool which provides an easy-to-use interface to a comprehensive, integrated suite of analytical tools for rapid in silico analysis of microarray, ChIP-seq and RNA-seq data and/or lists of genes. It also provides access to the novel analytical framework RMaNI, and to the widely used WGCNA tool for inference and analysis of regulatory networks using microarray data. Our web server makes available a set of tools and analytical workflows that would otherwise be challenging for non-technical users to install and apply. In future, we will integrate more tools and workflows to meet the distinct needs of researchers confronting the complexity of data.

 

Analytics framework

INsPeCT is a web-application with Drupal CMS front end and the complete application is running on a Linux high-performance computing (HPC) cluster.
